Documentation & Usage

Code Preprocessing

MAD IDE automatically handles CTSS abbreviations such as W'R for WHENEVER, expanding them before transpilation. Simply type your modern shorthand and the IDE will pad your labels to column 6, and format your statements from column 12 according to legacy IBM specifications.

Building The Compiler

To use the transpilation pipeline, you must have make and a C compiler (gcc or clang) installed. In the IDE, go to Tools -> Build MAD Compiler to initialize the backend.

Syntax Highlight

Built-in PyQsciScintilla lexer tailored specifically for MAD, featuring customizable keywords, strings, and comment stylings. Switch between light and dark modes easily via the top toolbar.
